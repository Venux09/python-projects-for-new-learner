# 新しい学習者向け Python プロジェクト 🐍

これらは、独学プログラマーとしてゼロからPythonを学びながら構築したプロジェクトです。

すべてのプロジェクトが何か新しいことを教えてくれました — エラー処理、
モジュール、API、ロジック構築など。

## プロジェクト:

### 🎮 ゲーム
- **アドベンチャーゲーム** — 複数のエンディングを持つテキストベースのストーリー
- **クイズゲーム** — スコア追跡を備えた複数選択クイズ
- **ミリオネアになりたい人** — 賞金付きターミナルクイズゲーム
- **数字当てゲーム** — ランダム数字当てゲーム
- **ロックペーパーシザース** — コンピューターと対戦
- **サイコロ振りゲーム** — マルチプレイヤーサイコロゲーム

### 🛠️ ツール
- **計算機** — エラー処理を備えた基本操作
- **アラームクロック** — カウントダウンタイマーと音声アラート
- **水飲みリマインダー** — デスクトップ通知リマインダー
- **水飲みリマインダー** — 水分補給を保つ通知
- **パスワード生成器** — 暗号化されたセキュアなパスワード
- **PDF マージャー** — 複数のPDFを1つにマージ
- **ナレッジアプリ** — 年齢ベースのURL リダイレクター
